# Risk Attributes

Every risk shall define:

- Risk ID

- Category

- Parent Category

- Description

- Severity

- Likelihood

- Impact

- Confidence

- Version

- Status

# Extension Rule

New risks shall extend existing categories whenever possible.

Duplicate risks are prohibited.

---

# Explainability Rule

Every enterprise risk assessment shall reference one or more canonical risk concepts.

# Golden Rule

Risk is a structured enterprise concept.
