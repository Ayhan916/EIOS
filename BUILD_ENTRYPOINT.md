# Purpose

This document defines exactly how any AI system shall start working on EIOS.

The purpose of this document is **execution sequencing**, not governance.

This document shall never redefine:

* project mission
* enterprise architecture
* governance rules
* terminology
* authority hierarchy

Those responsibilities belong to their respective authoritative artifacts.

If any contradiction exists, the higher-authority artifact shall prevail.


Do not start implementation before following this procedure.
