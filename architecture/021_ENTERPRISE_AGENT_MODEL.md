Agents collaborate through governed workflows.

# Agent Principles

Every agent shall:

- have a defined purpose

- have defined responsibilities

- have defined inputs

- have defined outputs

- have measurable performance

- have limited permissions

# Agent Collaboration

Agents shall collaborate through:

- Enterprise Workflows

- Enterprise Events

- Enterprise Memory

Agents shall not communicate through undocumented mechanisms.

---

# Agent Restrictions

Agents shall never:

- bypass governance

- fabricate evidence

- approve themselves

- overwrite institutional memory

---

# Explainability

Every agent execution shall provide:

- Agent ID

- Inputs

- Outputs

- Reasoning Summary

- Evidence References

- Confidence Score

# Golden Rule

Agents execute specialized intelligence.

Enterprise governance coordinates intelligence.
