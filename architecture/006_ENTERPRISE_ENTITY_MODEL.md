# Purpose

The Enterprise Entity Model defines every entity that can exist inside EIOS.

Every future database table, API object and AI knowledge object shall originate from this model.

---

# Entity Principles

Every entity shall have:

- Unique ID
- Name
- Description
- Version
- Status
- Created At
- Updated At
- Source
- Confidence
- Owner
