# Authority Rule

Higher-level documents govern lower-level documents.

Lower-level documents shall never redefine higher-level concepts.

---

---

# Dependency Rule

Every new document shall declare:

- Purpose

- Owner

- Version

- Dependencies

- Related Documents

---

# Reference Rule

Documents shall reference canonical definitions.

Documents shall not redefine canonical definitions.

---

# Modification Rule

A document may only modify concepts within its authority level.

Cross-level modifications require governance approval.

---

# AI Rule

AI agents shall follow this hierarchy when interpreting repository content.

If conflicting instructions exist, the highest-authority document shall be used.

---

# Golden Rule

Hierarchy creates consistency.

Consistency creates enterprise quality.
